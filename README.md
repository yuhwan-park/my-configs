## My Custom configs

### ESLint v9 + Prettier

**required**

```bash
yarn add -D eslint@^9.13.0 @eslint/js@^9.13.0 typescript-eslint eslint-plugin-prettier eslint-config-prettier globals
```

**for React**

```bash
yarn add -D eslint-plugin-jsx-a11y eslint-plugin-react eslint-plugin-react-hooks
```
